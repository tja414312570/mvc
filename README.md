# plugin.mvc MVC组件
特点
* CRUD方式请求（@RequestMapping，@GetMapping等）
* 参数验证 （基于jsr303 1.0.2 版本）
* multiForm表单支持
* 文件上传支持
* 内置静态资源支持（tomcat）
* 无状态token认证（支持方法级别和url级别的认证）
* 可扩展参数验证支持
* 可扩展参数类型功能
* 可扩展调配器
* 可扩展响应工具
* 路径变量
* 多种内置响应解析工具（json，文件等，响应状态）
